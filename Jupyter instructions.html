Welcome to Jupyter Notebook!

1. What is Jupyter Notebook?

Jupyter Notebook is an interactive computing environment that enables users to create notebook documents that include live code, interactive widgets, plots, narrative text, equations, and images. It's a powerful tool for data analysis, visualization, and exploratory programming.

2. How It Works

Jupyter Notebook works by creating and managing documents called "notebooks". These notebooks are a combination of text and code cells. Text cells allow you to write and format text using Markdown, while code cells allow you to write and execute code. When you run a code cell, the notebook's kernel executes the code and returns the output back to the cell below the code.

3. Basic Keyboard Commands

Here are some basic commands to get you started:

  • Shift + Enter: run the current cell and move to the next one.
  • Ctrl + Enter: run the selected cells.
  • Alt + Enter: run the current cell and insert a new one below.
  • a: insert a new cell above the current cell (in command mode).
  • b: insert a new cell below the current cell (in command mode).
  • dd: delete the current cell (press d twice in command mode).

To enter command mode, press Esc. To go back to edit mode, press Enter while a cell is selected.
